# FastAPI Scraper Server

A backend service built with FastAPI to scrape and process data, integrated with PostgreSQL and Dockerized for easy deployment.

---

## Features

- FastAPI for asynchronous API endpoints  
- PostgreSQL database for data persistence  
- Adminer for database management via web UI  
- Docker Compose setup for all components  

---

## Prerequisites

- Docker & Docker Compose installed  
- (Optional) Python 3.10+ for local development  
- Virtual environment (venv or similar) for local development  

---
